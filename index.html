    • üéØ Getting Started

    Initialize a New Repository

    git init
# Initialize a new local git repository

    Clone an Existing Repository

    git clone https://github.com/username/repository.git
# Clone via HTTPS

git clone git@github.com:username/repository.git
# Clone via SSH

git clone https://github.com/username/repository.git my-folder
# Clone into a specific folder

    üíª Basic Commands

    git status     # Check working tree status
git add .       # Stage all changes
git commit -m "message"  # Commit with message
git push        # Push to remote
git pull        # Pull from remote

    üåø Branching & Merging

    git branch           # List branches
git branch new-branch   # Create branch
git checkout branch-name  # Switch branch
git switch branch-name    # Switch branch (newer)
git checkout -b new-branch  # Create and switch
git merge branch-name     # Merge branch
git branch -d branch-name  # Delete branch

    üåê Remote Repositories

    git remote -v        # List remotes
git remote add origin   # Add remote
git fetch          # Fetch changes
git pull           # Fetch and merge
git push           # Push to remote
git push -u origin main  # Push and set upstream

    ‚öôÔ∏è Configuration

    git config --global user.name "Your Name"        # Set username
git config --global user.email "email@example.com"  # Set email
git config --list                                  # List all settings
git config user.name                              # Get specific setting

    üìú Viewing History

    git log                    # View commit history
git log --oneline        # Condensed history
git log --graph          # Visual graph
git diff                  # Show changes

    ‚Ü©Ô∏è Undoing Changes

    git reset HEAD~1        # Undo last commit
git reset --hard         # Discard all changes
git revert commit-id     # Revert specific commit
git checkout -- file     # Discard file changes

    üì¶ Stashing

    git stash                # Stash changes
git stash list           # List stashes
git stash pop            # Apply last stash
git stash drop           # Delete stash

    üè∑Ô∏è Tags & Releases

    git tag                  # List tags
git tag v1.0.0           # Create tag
git push origin v1.0.0   # Push tag
git tag -d v1.0.0        # Delete local tag

    üîß Advanced Commands

    git rebase main          # Rebase onto main
git cherry-pick commit-id  # Apply specific commit
